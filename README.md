# WarthunderWiki-Techtree-Costs
Userscript for wiki.warthunder.com.
Replaces the BR with RP and SL costs and adds Total Costs for Ranks and Trees

## Installation
### Option 1:
    1. Install Tampermonkey
    2. Open [this Link](https://github.com/NicolasNonsense/WarthunderWiki-Techtree-Costs/raw/main/TechTreeCost/WarthunderWiki-Techtree-Costs.user.js) (Tampermonkey should catch it automatically)
    3. press Install

### Option 2: 
    1. Install Tampermonkey
    2. Create a new Script
    3. Copy and paste the WarthunderWiki-Techtree-Costs.js code into the script

## Usage
- On your first visit to a tech tree, all costs will show as **0** .
- The script then loads costs vehicle by vehicle.
- If nothing updates after ~10 seconds, **refresh the page**.
- Once loaded, values are **cached** and appear **instantly** on future visits.

## Features
- Replaces BR values with **RP** and **SL** costs on War Thunder Wiki tech trees
- Adds **total costs** per **rank** and for the **entire tree**
- All Values are **cached** results for faster future visits
- Planned: Reset button to refresh a tree when costs change (currently requires manually deleting the cached value)


## Requirements
- A browser with a userscript manager:
  - **Tampermonkey** (recommended)
  - Violentmonkey / Greasemonkey should also work
